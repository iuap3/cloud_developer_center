# 云开发者中心租户开通

  打开云开发者中心(https://developer.yonyoucloud.com),注册用户并登录，点击入驻申请。

</div>

  在入驻界面中输入申请信息,点击提交。

</div>
<p align="center">图 2</p>

  提示成功后回到开发者中心,刷新界面,可以看到审核状态。
