# 日志管理

## 简介
日志管理收集应用运行日志，并提供简单、易用的日志分析、管理功能，用户可以利用它对日志进行集中管理和准实时搜索、分析，使用搜索、可视化等简单易用的功能，帮助用户进行线上业务的准实时监控、业务异常时及时定位原因、排除故障、程序研发时跟踪分析Bug等功能。本文介绍如何使用日志管理功能。
